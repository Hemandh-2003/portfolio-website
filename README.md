Installation & Setup

Clone the repository:
git clone https://github.com/Hemandh-2003/portfolio-website.git
cd portfolio-website

Install dependencies:
npm install

create .env file in root directory
MONGO_URI=your_mongodb_connection_string
PORT=5000

Start the server:
npm start

http://localhost:5000
